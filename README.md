## Project List

### 1. Data Cleaning
- Focus: Removing duplicates, handling blanks, fixing formats, using `TRIM`, `PROPER`, and `Find & Replace`.
- Tools: Excel formulas + Data tab tools
- Goal: Prepare messy raw data for analysis

### 2. Creating Basic Charts
- Focus: Column charts, pie charts, line graphs
- Goal: Visualize simple data trends and comparisons
- Use case: Sales by product, monthly expenses, etc.

### 3. Creating Advanced Charts
- Focus: Combo charts, dynamic charts with slicers, sparklines, waterfall and funnel charts
- Goal: Enhance storytelling and interactivity in Excel reports

### 4. Creating a Simple Dashboard with Excel
- Focus: Combine charts, pivot tables, filters, and slicers into one view
- Features: Drop-downs (data validation), `IF`, and Pivot Tables
- Goal: Build a user-friendly executive dashboard using only Excel

### 5. Premier-Player-23-24
- Focus: Analyzing Premier League player data (2023/24 season)
- Skills Used: Sorting, conditional formatting, ranking, custom functions
- Example Insights:
  - Top scorers and assisters
  - Player stats by position and team
